# 更新日志 (CHANGELOG)

## [v0.0.1] - 初始测试版本

### ✨ 新功能

* 实现基础 **Obsidian 日记自动修改功能**
* 支持通过指令更新日记内容
* 支持 AI 根据用户指令修改已有日记
* 支持在 AstrBot 插件系统中运行

### 🛠️ 当前实现

* 基础命令解析（如：`更新测试日记: 内容`）
* 调用 LLM 对日记内容进行修改
* 日记文件读写逻辑

### ⚠️ 已知问题

* LLM 在修改日记时可能会 **自动扩展无关信息**
* 例如仅提供人名时，可能会推测人物信息并生成额外内容
* Prompt 控制仍在优化中

### 🚧 开发计划

以下功能正在开发中：

* 插件配置文件 `_conf_schema.json`
* 支持自定义 Prompt 配置
* 支持日记记录 Prompt
* 支持日记修改 Prompt
* 限制 AI 输出长度
* 优化 AI 修改逻辑，减少无关扩写
